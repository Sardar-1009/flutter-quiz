# Flutter Quiz App

Приложение-викторина (Quiz) для проверки знаний по Dart и Flutter. Проект выполнен с использованием архитектурного паттерна **MVVM** (Model-View-ViewModel) и обеспечивает чистое разделение логики и интерфейса.

## Особенности
- **Архитектура MVVM**: Четкое разделение ответственности.
- **Динамические вопросы**: Загрузка данных из локального JSON-файла.
- **Умная выборка**: Приложение автоматически выбирает сбалансированное количество вопросов из разных категорий (Dart, Widgets, Architecture и др.).
- **Подсчет результатов**: Детальный экран итогов с процентом правильных ответов.
- **Поддержка платформ**: iOS, Android, macOS и Web.

## Требования
- **Flutter**: `>= 3.0.0`
- **Dart**: `>= 3.0.0`

## Инструкции по запуску

### 1. Склонируйте репозиторий
```bash
git clone <url-репозитория>
cd flutter_quiz_app
```

### 2. Установите зависимости
```bash
flutter pub get
```

### 3. Запустите приложение
Вы можете запустить приложение на любой доступной платформе:

**Для macOS:**
```bash
flutter run -d macos
```

**Для браузера (Chrome):**
```bash
flutter run -d chrome
```

**Для Android/iOS:**
```bash
flutter run
```

## Структура проекта
- `lib/models/`: Модели данных.
- `lib/services/`: Сервисы (загрузка данных).
- `lib/viewmodels/`: Логика приложения и управление состоянием.
- `lib/views/`: Экранные интерфейсы и виджеты.
- `assets/data/`: JSON-файл с вопросами.
